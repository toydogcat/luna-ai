# luna-ai
The Home Page for Luna.

This repository hosts the centralized **Luna AI Hub**, constructed with React + Vite and styled for premium UI aesthetics.

## Core Infrastructure
- **React 18 + Vite** for instant builds and frontend agility.
- **Firebase Hosting** for production edge delivery.
- **Firebase Analytics** for automated usage reporting & event tracking.

## How to Develop Locally
```bash
npm install
npm run dev
```

## How to Build & Deploy
```bash
npm run build
firebase deploy --only hosting
```
