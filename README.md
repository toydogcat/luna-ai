# luna-ai
The Home Page for Luna.

This repository hosts the centralized **Luna AI Hub**, constructed with React + Vite and styled for premium UI aesthetics.

## Core Infrastructure
- **React 18 + Vite** for instant builds and frontend agility.
- **Firebase Hosting** for production edge delivery.
- **Firebase Analytics** for automated usage reporting & event tracking.

## How to Develop Locally
```bash
npm install
npm run dev
```

## How to Build & Deploy
```bash
npm run build
firebase deploy --only hosting
```

## 💡 如何安裝 Luna AI Web App (PWA 指南)

本專案支援 **PWA (Progressive Web App)** 規範，您可以將其安裝為獨立 App 以獲得最佳體驗：

### 📱 手機裝置 (iOS / Android)
1. 在 Chrome、Brave 或 Safari 打開網站 [luna-ai-b7862.web.app](https://luna-ai-b7862.web.app)。
2. 點擊瀏覽器的 **「分享」** 圖示 或 **「功能表」(三個點點)**。
3. 選擇 **「加到主畫面」** (Add to Home Screen)。
4. 完成後，您的桌面就會出現專屬的高質感 Luna AI 圖示，點擊即可全螢幕開啟！

### 💻 電腦裝置 (Desktop)
1. 使用 Chrome / Edge / Brave 等核心瀏覽器打開網站。
2. 在網址列右方會看見一個 **「安裝應用程式」** 的小圖示 (+)。
3. 點選並確認安裝，應用程式將會常駐在您的應用程式清單中。
